# Monthly Sales Summary (2024)

## Overview
This Excel workbook provides a detailed summary and analysis of monthly sales performance for the year 2024. It includes raw sales data, summarized reports using pivot tables, and a visual dashboard designed to present key performance insights in an easy-to-understand format.

## File Structure

### 1. Data
The **Data** sheet contains the base dataset used for all analyses.  
It includes transaction-level sales information such as:
- **Date / Month**
- **Product or Category**
- **Region / Branch**
- **Units Sold**
- **Sales Amount**
- **Profit / Margin**

This sheet serves as the primary input for the pivot tables and charts.

### 2. PivotTables
The **PivotTables** sheet summarizes the raw data to highlight major trends and performance indicators.  
Typical summaries include:
- Total sales and profit by **month**
- Sales comparison by **region**
- Top-performing **products or categories**
- Average profit margins and sales growth

The purpose of this sheet is to make it easier to identify sales trends and performance gaps throughout the year.

### 3. Dashboard
The **Dashboard** sheet provides a visual representation of key metrics and trends derived from the PivotTables.  
It is designed for quick insights and includes:
- Sales and profit charts over time  
- Regional performance comparisons  
- Best- and worst-performing categories or products  

> *Note:* The dashboard relies on dynamic connections to the PivotTables sheet, so all data updates automatically when the underlying data changes.

## Purpose
The workbook is intended to help monitor sales performance, analyze monthly trends, and support data-driven decision-making. It is suitable for business reviews, academic reports, or internal performance tracking.
